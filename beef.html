BeEF is a tool that generates a link that you can send to someone and gain complete access to their web browser and implement Metasploit.

Chapter One: Installing BeEF

To install from your OS's package manager, know that the package is called beef-xss. To install from git, type 'git clone https://github.com/beefproject/beef.git'

Chapter Two: Using BeEF

You can just type 'sudo beef-xss' and BeEF will be started on your machine. To send it to somebody on your network, you need to know your IP address. You can get this by typing 'sudo ifconfig' This should return your IP address on the second line, right after 'inet' and before 'netmask'. To get to the login page, go to this address: '(Your IP):3000/ui/panel'. The username + password are both beef, and the hook url (the url that you send to your victim) is:'(Your IP):3000/demos/butcher/index.html'.

To start BeEF with the github package, type 'cd beef', and then type these five commands:
sudo (OS package manager installation method, for Debian its apt-get install) ruby
./install
./update-beef
gem install bundle
bundle install
But first, we need to change the default username and password by typing 'vim config.yaml', then scrolling down and typing 'i', which allows you to edit, then deleting the default credentials and changing them. When you are finished, type ":wq". This leaves and saves the file. Finally, you can start BeEF by typing './beef'. You will know what to do from there.
