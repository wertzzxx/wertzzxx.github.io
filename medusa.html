Medusa:

Photo taken from Geeks for Geeks: Here

Medusa is a password cracking tool that is used for Dictionary attacks.

Lets start!

Chapter One: Installing Medusa

To install Medusa, you need to use your OS application manager. (apt, pacman, etc.) For example: 'sudo apt-get install medusa' for Debian/Debian-based users.

Medusa should be installed, but you should always check. To make sure that Medusa is installed on your machine, type 'medusa' in your terminal. It should return instructions for how to use it.

Chapter Two: Using Medusa

Using Medusa is pretty simple, but it can be pretty daunting with all of the instructions needed.

You can either A, read all the instructions that it gives you, or B, just learn through this chapter.

First, you need to know your target. You need to get the host IP address from the target. You can figure this out by typing this in your terminal: 'dig (target website)' Replace target website with website url, just without 'http(s)://'. The IP address should be returned in the 'Answer Section'.

Second, you need to get a password file that contains all the possible passwords a user can have, or you can just type the passwords out instead of having a file containing them. My recommendation is rockyou.txt if you don't have any specific password files. You can download it from this link: Here. You also need to have a username.

Lastly, we need to put all of the information together in tags. The tags that we need to know are:

  1. -h: the host IP
  2. -u: username
  3. -p: single password
  4. -P: password file
  5. -M: module
  6. -n: port number

Some hosts may use false positives, so you will get an incorrect password. Or, they will disable less secure applications to attempt to sign in.
